# Ecommerce

## How to run the project
1. start mysql service
2. create a mysql database called `stripe_integration`
3. start `redis-server`

### `server`
```
cd server
yarn
yarn watch
yarn dev
```
### `web`
```
cd web
yarn
yarn dev
```
